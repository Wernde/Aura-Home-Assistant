# AURA Home Build Roadmap

## Alpha 0.1 — Living visual and simulated home

- Animated holographic AURA face
- Wall-screen layout
- Touch interactions
- Voice and typed demo commands
- Simulated household state
- Local persistence

## Alpha 0.2 — Camera and voice hardware

- Camera permission and physical privacy controls
- Presence detection
- Basic gaze tracking
- Far-field microphone configuration
- Local wake-word prototype
- Improved viseme-driven speech animation

## Alpha 0.3 — Real intelligence

- OpenAI Realtime voice connection
- Responses API tool layer
- Conversation and approved household memory
- Specialist-agent routing
- Approval prompts for sensitive actions

## Alpha 0.4 — Real home connections

- Home Assistant integration
- MQTT event stream
- Confirmed device states
- Real scenes and automations
- Weather and calendar integrations

## Beta — Profiles and controlled autonomy

- Face and voice profiles
- Family permissions
- Personal calendars and reminders
- Proactive recommendations
- Per-capability autonomy levels
- Administrator console and mobile companion
